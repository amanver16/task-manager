# Task Manager Application

This repository is designed to teach Full Stack Development step by step using a Task Manager application as the example project.

The goal is to help beginners and less experienced developers learn gradually without being overwhelmed by too many concepts at once. Each module introduces new technologies and concepts while building upon knowledge gained in previous modules.

## Learning Objectives

By completing this roadmap, learners will gain hands-on experience with:

- HTML
- CSS
- Bootstrap
- JavaScript
- TypeScript
- Git & GitHub
- Angular
- Java
- REST APIs
- Spring Boot
- SQL & MariaDB
- JPA / Hibernate
- Application Testing
- Deployment
- Related Tools & Softwares
- Coding Best Practices

## Roadmap

The below folder structure covers basic to advanced Task Manager development step by step:

### 1. task-manager-html

Learn HTML fundamentals and build the basic structure of the Task Manager application:

- HTML Elements
- Forms
- Tables
- Semantic HTML
- Accessibility Basics

### 2. task-manager-html-css

Enhance the application with styling and responsive layouts:

- CSS Fundamentals
- Flexbox
- Grid
- Responsive Design

### 3. task-manager-html-css-bootstrap

Improve UI development using Bootstrap components and layouts:

- Bootstrap Grid System
- Forms
- Navigation
- Utility Classes

### 4. task-manager-html-css-bootstrap-javascript

Add interactivity to the application using JavaScript:

- Variables
- Functions
- DOM Manipulation
- Events
- Local Storage

### 5. task-manager-html-css-bootstrap-typescript

Introduce TypeScript and improve code maintainability:

- Types
- Interfaces
- Classes
- Modules
- Async Programming

### 6. task-manager-git-github

Learn version control and collaboration workflows:

- Git Basics
- Branching
- Merging
- Pull Requests
- GitHub Repositories

### 7. task-manager-angular

Convert the frontend into a modern Angular application:

- Components
- Services
- Routing
- Forms
- Dependency Injection

### 8. task-manager-java

Learn Java basics before jumping into backend development:

- Core Java Fundamentals
- Object Oriented Programming (OOPS Concepts)
- Collections
- Exception Handling
- Java 8+ Features


### 9. task-manager-rest-api-concepts

Understand how frontend and backend applications communicate:

- HTTP
- REST APIs
- JSON
- Request & Response Lifecycle
- Status Codes

### 10. task-manager-springboot

Build backend APIs using Spring Boot:

- Controllers
- Services
- Dependency Injection
- Exception Handling

### 11. task-manager-angular-springboot

Integrate Angular frontend with Spring Boot backend APIs:

- HTTP Client
- API Consumption
- Error Handling

### 12. task-manager-sql-mariadb

Learn relational database concepts and SQL using MariaDB:

- Database Design
- CRUD Operations
- Joins
- Indexes
- Relationships

### 13. task-manager-angular-springboot-jpa

Persist application data using JPA and MariaDB:

- Entities
- Repositories
- Relationships
- Pagination
- Query Methods

### 14. task-manager-testing

Unit testing the application components:

- Unit Testing
- API Testing
- Frontend Testing

### 15. task-manager-deployment

Prepare and deploy the application:

- Production Builds
- Environment Configuration
- Deployment Basics

### 16. task-manager

Final production-ready Task Manager application that combines everything learned throughout the roadmap.

### 17. best-practices

Although the entire roadmap will follow these best practices but this module is simply dedicated towards coding best practices:

- Language Specific Naming Conventions
- Code Formatting
- Using Code Quality Tools
- Proper Folder/File Structure
- Organized Imports
- Addressing Unused and Deprecated Code
- Code Documentation and Comments

## Recommended Learning Approach

- Complete modules in sequence.
- Focus on understanding concepts before moving to the next module.
- Practice by extending the Task Manager application at every stage.
- Review previous modules whenever a concept feels unclear.
- Build small enhancements independently to reinforce learning.

The objective is not just to build a Task Manager application, but to understand the complete Full Stack Development lifecycle from application creation to deployment.
