# Task Manager with HTML and CSS

In this lesson we will learn to add styles to our task manager application using CSS.

## What is CSS?
CSS stands for **Cascading Style Sheets**. It is a language used to describe the presentation of a web page written in HTML. CSS allows you to control the layout, colors, fonts, and overall appearance of your website.

### Types of CSS
- **Inline CSS**: Styles are applied directly to an HTML element using the `style` attribute. (Example: `<p style="color: red;">This is a red paragraph.</p>`).
- **Internal CSS**: Styles are defined within a `<style>` element in the `<head>` section of the HTML document. (Example: `<style> p { color: blue; } </style>`).
- **External CSS**: Styles are defined in a separate `.css` file and linked to the HTML document using the `<link>` element. (Example: `<link rel="stylesheet" href="styles.css">`).

## CSS Topics covered in this project:

- **Selectors**: CSS selectors are used to target HTML elements and apply styles to them. (Example: `p { color: green; }`).
- **Colors and Backgrounds**: CSS allows you to set the color of text and the background of elements. (Example: `body { background-color: lightgray; }`).
- **Typography**: CSS provides properties to control the font, size, weight, and style of text. (Example: `h1 { font-family: Arial, sans-serif; }`).
- **Box Model**: The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of margins, borders, padding, and the actual content. (Example: `div { margin: 10px; padding: 20px; border: 1px solid black; }`).
- **Display Properties**: CSS display properties control how elements are displayed on the page. (Example: `span { display: inline; }`).
- **Flexbox**: Flexbox is a CSS layout module that provides an efficient way to lay out, align, and distribute space among items in a container. (Example: `div { display: flex; }`).
- **Grid**: CSS Grid is a powerful layout system that allows you to create complex and responsive grid-based layouts. (Example: `div { display: grid; grid-template-columns: 1fr 1fr; }`).
- **Responsive Design**: Responsive design is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. (Example: `@media (max-width: 600px) { body { background-color: lightblue; } }`).
